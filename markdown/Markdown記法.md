# Markdown記法



## これは何？

テキストをボールドやリスト表示、表形式として簡単に打ち込むことができる記法。

記法は簡単で読みやすく、GitHubのREADMEや、各種Webサービス、アプリで利用できる。

- 書きやすい
- 覚えやすい

## 微妙なところ

- エディタによって方言があり、変換してくれたりしてくれなかったりする
